01

연구 배경 및 필요성

이커머스 시장의 성장과 피킹 비용 문제
피킹 작업 비용
  • 총 운영비용의 50% 이상을 피킹 작업이 차지
    (Zhuang et al., 2024)
무료배송 정책
  • 무료 배송 기준 설정 시 평균 주문 금액 약 2.5배 증가
  • 1건 주문당 구매 상품 수 증가
재고배치의 중요성
  • 처리 상품 종류 증가 → 작업자가 방문해야 할 지점의 수 증가
연구 필요성 — 4가지 핵심 목표
1
상품 네트워크 구조 확인 — 고객 구매데이터를 인접행렬로 변환하여 상품 네트워크 구조 시각화
2
집단 간 연결 강도 정량화 — 중심성 지표를 활용하여 집단 간 연결 강도 정량화
3
풀필먼트 센터의 피킹 효율 향상 — 네트워크 구조와 연결 강도를 재고배치 모형에 적용
4
성능 평가 — ABC 분석, 연관성 분석, SNA 분석 기반 재고배치 모형 비교 검증
02

선행연구 검토

ABC분석·연관성 분석·SNA 분석 비교
구분ABC 분석연관성 분석
분석 단위개별 상품 (빈도)상품 쌍 (동시구매)
주요 지표출고 빈도 / 금액지지도·신뢰도·향상도
방법출고 빈도 기준으로 상품을 A,B,C 등급 구분함께 주문되는 상품 집합 추출, 인접 위치에 배치
장점단순·직관적상품 간 관계 반영
한계상품 간 관계 무시대량 중복 규칙 생성, 규칙 간 관계 파악 곤란
대표 연구임우택(2014)박광현(2016), Pang(2017), Zhou et al.(2020)
SNA 분석의 차별성
네트워크 구성
  • 상품 전체를 하나의 네트워크로 구성
  • 직간접적인 연결 관계까지 반영
중심성 지표
  • 연결 중심성(Degree)
  • 매개중심성(Betweenness)
  • 근접중심성(Closeness)
분석 범위 확대
  • 개별 속성 기준(ABC)
  • → 상품 간 관계 패턴(연관규칙)
  • → 위상 구조(SNA)
03

연구 모형 및 방법론

연구 절차 (6단계)
1
데이터 수집·전처리 — 이커머스 의류·종합쇼핑몰·완구 주문 데이터(2개월), 클렌징 작업
2
탐색적 분석 — 주문 현황 및 주문 특성 파악을 통한 데이터 구조 탐색
3
데이터 처리 — 데이터 구조 변환: adjacency list → adjacency matrix
4
데이터 시각화 — 상품 네트워크 구조 확인, 중심성 비교(degree · closeness · betweenness)
5
모델링·적용 — 측정지표 정의(density, rack 방문 횟수), 기본모형·비교모형 설계
6
성능 비교 — 모형별 성능 비교: rack 방문 횟수, 이동거리
네트워크 모형 정의
네트워크 구성 요소
  • 노드: 상품
  • 엣지: 동일 주문에 포함되어 함께 주문된 관계
  • 가중치(aij): 두 상품이 동시에 포함된 주문 건수
⚠️ 논문 내 개념 혼재 발견

aij에 대해 '수량의 곱'이라는 개념과 '상품i·j가 동시에 포함된 주문 수'라는 개념이 혼재되어 있음. 가중치 정의의 명확화 필요.
